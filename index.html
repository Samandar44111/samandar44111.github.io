Trading Bot

Live Demo

A BTC/USDT paper-trading dashboard driven by an RSI + EMA strategy — updating live from the bot's own trades.

PythonRSI + EMALive Data

A Python bot runs an RSI + EMA strategy on a virtual machine, paper-trading BTC/USDT and pushing every trade to GitHub. This dashboard reads that data and visualises it in real time — no login, and no real money at risk.

What you'll see
  • Live BTC price, RSI, and daily profit / loss
  • Current position and account equity
  • A price chart with buy/sell markers, from 1H to all-time
  • Everything auto-refreshes every 30 seconds

Linear Equation Solver

Live Demo

Solve systems of 1–5 variables with step-by-step Gaussian elimination — type the numbers, get the answer.

JavaScriptLinear AlgebraStep-by-step

A clean math tool for solving systems of linear equations. Pick how many variables you have, enter the coefficients, and it solves them using Gaussian elimination with partial pivoting — then shows the full working if you want to follow along.

What you'll see
  • Support for 1 to 5 variables
  • An instant solution for each variable
  • An optional step-by-step breakdown
  • Example problems you can load with one tap
